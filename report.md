# Advanced Time Series Forecasting with Prophet and SHAP

Author: Dheenadhayalan

## Dataset
Seven-year daily synthetic sales dataset with multiple seasonalities and holidays.

## Model and Validation
Prophet model tuned using walk-forward cross-validation with RMSE evaluation.

## Explainability
SHAP analysis identifies trend, yearly seasonality, marketing spend,
promotions, and weekly seasonality as dominant contributors.
